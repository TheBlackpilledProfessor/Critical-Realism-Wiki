# Critical Realism Analyzer

A single-use topic analysis tool powered by AI language models. Enter any topic and receive a comprehensive, evidence-based analysis written in a natural, engaging style.

## Architecture

Pure client-side application with provider-agnostic AI integration. No backend, no database, no persistence - just you and the AI.

## Features

- **15 Specialized Analysis Modes:** Automatically detects topic type (academic, cultural, practical, social)
- **Real-time Streaming:** Watch AI-generated content appear word-by-word
- **Evidence-Based:** All claims cited with reputable sources
- **No Data Persistence:** Pure client-side, zero storage, maximum privacy
- **Category Detection:** Keyword-based classification across 15 categories
- **Responsive Design:** Works on desktop and mobile
